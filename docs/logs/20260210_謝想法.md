## 🛠️ 實作過程 / 決策分析

### 1. 標的選擇：西瓜 (Watermelon)
經過比較分析，最終決定以 **西瓜** 作為晶片測量對象，原因如下：
* **阻抗變化率最大 (High Dynamic Range)：** 生瓜與熟瓜的主要差異在於細胞壁是否崩解釋放游離水。根據資料，這種物理變化會導致阻抗值有顯著差異，對於類比前端 (AFE) 的訊號捕捉最為友善。
* **排除其他選項：**
    * **苦瓜/木瓜：** 可透過外觀顏色或觸摸硬度判斷，開發電子檢測儀器的「必要性」與「痛點」不足。
    * **哈密瓜：** 雖然經濟價值高，但網紋表皮導致接觸阻抗不穩，且實驗成本過高。

### 2. 核心技術：生物阻抗分析 (Bio-impedance Analysis, BIA)
* 不只是單純測量電阻，而是計畫導入 **BIA 技術**。
* **技術亮點：** 計畫設計「多頻率訊號產生器 (Multi-frequency Generator)」，利用不同頻率響應來分析果肉特性（低頻測細胞外液、高頻測細胞結構）。這將是與老師討論的重點技術。

### 3. 測量方法：四線式測量 + AC 激發
* **消除誤差：** 採用 **開爾文測量法 (Kelvin Sensing / 4-wire)**，利用 Force/Sense 分離架構，消除探針接觸電阻與導線電阻的影響。
* **防止電解：** 激發訊號必須使用 **交流電 (AC)**，避免直流電導致果肉電解產生氣泡，造成探針生鏽與數值飄移。

### 4. 晶片架構草案 (Block Diagram Draft)
目前的系統規劃包含以下區塊：
1.  **類比前端 (Analog Front-End)：** 負責產生 AC 訊號與訊號放大。
2.  **ADC / 訊號處理：** 將類比訊號轉換為數位數值。
3.  **校正電路 (Calibration)：** 針對不同大小的西瓜進行歸零或正規化運算。
4.  **輸出介面：** LED 顯示結果。

## ❓ 遇到的問題 / 待確認事項
1.  **BIA 的具體頻率範圍：** 西瓜的特徵頻率大約落在多少 Hz？需要查閱更多論文或進行初步掃頻實驗。
2.  **校正邏輯：** 關於「歸零」機制，若探針間距固定，是否測量「電阻率」即可忽略西瓜體積大小的影響？這點需要進一步驗證。
