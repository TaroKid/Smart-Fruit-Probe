## 研究成果
1. 探針必須刺入瓜皮接觸內部果肉才能有效量測阻抗。這表示測試過後水果必定無法長時間保存。
2. 如果選用木瓜、芒果、哈密瓜等水果，過軟的肉質可能對測量造成影響，需要進一部驗證。另外木瓜和芒果表皮損傷時分泌的乳汁可能會被探針帶進果實內部造成接觸不良和探針污損，此為兩種水果的缺點。
3. 若探針過長，且水果中空，則可以考慮探針的插入方向，以中醫埋線手法進行探針插入。
4. 苦瓜是很好的選擇，因其生熟程度幾乎可靠肉眼和觸覺準確判斷，但科研價值也可能因此較低；不新鮮的蓮霧內部可能已經海綿化，電阻理論上很大，所以以蓮霧作為題材也是意個不錯的選擇，然而破壞性的檢測仍然是需要解決的痛點，因為蓮霧易腐更嚴重。
5. 只以單顆LED定生死的誤差太高，如果改用交流信號輸入來測量，並用放大器作為buffer把水果的高阻抗信號轉成低阻抗信號從而準確測量，不僅可以得到更精細的數值，也可避免直流電造成水果內部受到電解而使數據失真。
6. 若以交流信號測量，可以用頻率響應尋找最適合的測量信號頻率。
7. 除了熟度的測量，也可順便驗證甜度和熟度的相關性。
